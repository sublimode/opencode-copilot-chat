# 📸 Screenshots & Demo Assets

Place screenshot and GIF files here. They'll be referenced from the README and landing page.

## 📏 Naming convention

| File | What to capture | Suggested size |
|---|---|---|
| `hero.png` | Main hero image for landing page | 1200×630 (OG image) |
| `model-picker.png` | Copilot Chat model picker showing OpenCode models | 800×500 |
| `usage-statusbar.png` | Status bar `Go: 27%·62%·75%` indicator | 400×80 |
| `thinking-picker.png` | Thinking Effort per-family dropdown | 600×400 |
| `diagnostics-report.png` | Markdown diagnostics report | 800×600 |
| `agent-mode-demo.gif` | Agent Mode editing files with an OpenCode model | 800×500 |
| `quick-start.gif` | Full quick-start flow animated | 800×500 |

## 🛠️ Tools

- **macOS screen recording:** Cmd+Shift+5 → export to GIF via [Gifski](https://gif.ski/)
- **Cross-platform GIF:** [LICEcap](https://www.cockos.com/licecap/) or [Peek](https://github.com/phw/peek)
- **OG image generator:** [og-image.xyz](https://og-image.xyz/)

## 📤 Contributing screenshots

1. Take the screenshot/recording.
2. Drop the file here with the naming convention above.
3. Open a PR — I'll wire it into the README and landing page.

See [`CONTRIBUTING.md`](../../CONTRIBUTING.md) for full guidelines.
